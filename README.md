# 桌面自动化智能体系统

这是一个基于视觉理解的桌面自动化智能体系统，能够通过分析屏幕截图执行用户登录和网页搜索等任务。系统结合了大型视觉语言模型（Qwen2.5-VL）和桌面自动化技术，实现智能的桌面操作。

## 功能特点

- **智能视觉理解**：使用Qwen2.5-VL模型分析屏幕内容
- **自动化操作**：支持点击、输入、滚动等桌面操作
- **多任务支持**：实现用户登录和网页搜索功能
- **可视化界面**：提供直观的Gradio用户界面
- **错误恢复机制**：内置重试和错误处理逻辑
